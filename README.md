# Plant Disease Detection System

Simple project to train and serve a plant disease detection model.

## Contents
- `app.py` - Flask app to serve predictions
- `Train_plant_disease.ipynb` - Notebook used for training
- `training_hist.json` - Training history

## Quickstart
1. Create a virtual environment: `python -m venv .venv`
2. Activate it and install requirements
3. Run `python app.py` to start the server
